- The full, detailed commit history from the entire Part 1 development phase and early Part 2 phase was permanently lost.
- I attempted to recover the history using `git reflog` and consulted AI assistants for troubleshooting, but the full commit history could not be restored.

## 🚀 How to Run the Application
### Part 1 (Vanilla JS/TS)
1.  Navigate to the `Part1/` directory.
2.  Open `index.html` in any modern web browser to run the application.

### Part 2 (Angular)
1.  Ensure Node.js and npm are installed on your system.
2.  Navigate to the root directory of the repository.
3.  Run `npm install` to install all required dependencies.
4.  Run `ng serve` to start the Angular development server.
5.  Open `http://localhost:4200/` in your browser to access the full inventory system.
